Shopping Cart Portal

Description: Created a online shopping cart portal using MVVM architecture. A shopping cart is basically a list that contains products selected by the user while he shops. When the user is finished shopping, he will usually examine the list to double-check that the items, quantities, and prices are correct. If he finds any errors he should be able to edit the list. Once he is ready, he should be able to check out. The checkout process involves an exchange of information that results in a sale.

Responsibilities:

  • Created Shopping Cart Portal using HTML 5, CSS 3,Angular JS, JQuery, Bootstrap
  • Created a single page application using MVVM architecture where a user can search, add, delete products to his shopping cart.
  • Created custom directives and filter for the product page.
  • Added PayPal and Google Wallet payment service for checkout
Environment:

HTML, CSS 3, Angular JS , JQuery, Bootstrap.
